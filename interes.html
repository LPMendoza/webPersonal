Temas de Interes

Desarrollo Front-End

El desarrollo Front-End es un tema que me interesa mucho, el como hacer y diseñar una aplicación de tal modo que sea funcional y atractiva para el usuario, además de que sea fácil de usar y que se adapte a diferentes dispositivos

Tecnologías

Hay diferentes tecnologías con los cuales desarrollamos la parte de Front-End en web como son HTML5, CSS3, JavaScript, React y Bootstrap

HTML5, CSS3 y JavaScript

Estos tres lenguajes son la triada perfecta para la creación de una página web en la parte de Front-End, HTML5 nos da la estrucutra, CSS3 los estilos de la pagina web, y JavaScript funcionalidad

React

Un framework el cual nos facilita la creación de paginas web de una sola página, por lo tanto los datos se actualizan en tiempo real sin tener que recargar la página, esto da más facibilidad de uso al usuario

Bootstrap

Un framework que provee de componentes y estilos ya prefabricados para nuestra página web, ademas de que facilita la creación de una página web responsiva.

Metodologías ágiles

Las metodologías ágiles asi como todo lo que tiene que ver con la gestión de proyectos, es un tema de mi interes, mis metodologías ágiles preferidas son Scrum y Kanban, ya que combinando estas dos se puede lograr una buena gestión.
